## Superspeed.sh<sup>[1](#脚注1)</sup>
使用全国各地三大运营商<sup>[2](#脚注2)</sup>的 speedtest 测速节点进行全面测速<sup>[3](#脚注3)</sup>。

## 使用方法
```
bash <(curl -Lso- https://raw.githubusercontent.com/myxuchangbin/superspeed/master/superspeed.sh)
```
